## Description

API based on the [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository, providing the backend support for a Twitter bot that posts the daily Catholic readings to Twitter in a thread.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
